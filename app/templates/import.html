⚠️ Import Disabled

Historical data import is disabled in production for security reasons.

To Enable Import

If you need to import historical data, temporarily enable the feature:

  1. Set the environment variable: ENABLE_IMPORT=true
  2. Redeploy or restart the application
  3. Perform your import
  4. Remove the environment variable and redeploy to disable again

Azure CLI Example

# Enable import temporarily
az webapp config appsettings set \
  --name app-puerhumidity \
  --resource-group rg-puerhumidity \
  --settings ENABLE_IMPORT=true

# After importing, disable again
az webapp config appsettings delete \
  --name app-puerhumidity \
  --resource-group rg-puerhumidity \
  --setting-names ENABLE_IMPORT

How to Generate a PAT

  1. Go to SmartThings My Tokens
  2. Click "Generate new token"
  3. Name it something like "PuerHumidity Import"
  4. Select scopes: Devices (Read) and Locations (Read)
  5. Click "Generate token" and copy it
  6. Paste it above and click "Import Data"
